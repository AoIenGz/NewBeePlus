- 功能测试：覆盖手动检测、产线模式、AI检测、图片纠偏、历史记录、统计分析、数据导出等全部功能

## 三、接口测试用例（API）

### API-01：POST /api/detection/analyze

| 项目 | 内容 |
|------|------|
| 用例编号 | API-01 |
| 接口地址 | POST /api/detection/analyze |
| 请求体 | `{ "image": "data:image/jpeg;base64,/9j/4AAQ..." }` |
| 预期响应 | 200, `{ "success": true, "grade": 3, "gradeMethod": "color", "gradeConfidence": 0.85, "defects": {...}, "position": {...}, "isPass": true }` |
| 验证点 | success=true; grade 为 1-5 或 null; defects 对象包含 isDamaged/isStained/isWrinkled; position 对象包含 isCorrect/x/y/deviation |

### API-02：GET /api/history

| 项目 | 内容 |
|------|------|
| 用例编号 | API-02 |
| 接口地址 | GET /api/history?page=1&limit=10 |
